# Changelog

## 0.2.0 - 2026-06-09

### Added

- Added a Japanese study report template with clearer learning, validation, and portfolio sections.
- Added a Linux `grep`/`find` practice lab with a sample log file.
- Added a networking `ping` vs `curl` practice lab.
- Added a Korean beginner quickstart guide.
- Added an AWS S3 beginner lab template with credential-safety guidance.
- Added an AWS S3 evidence example using placeholder and redacted values.

### Changed

- Improved README discoverability by linking directly to docs, labs, and templates.
- Updated the roadmap to reflect completed v0.2.0 learning resources.

### Safety

- Cloud documentation uses placeholders such as `<bucket-name>` and `<region>`.
- No AWS credentials, account IDs, private URLs, or real infrastructure details are included.
- Learners are reminded to clean up test resources that may incur cost.
