# Short Sharing Drafts

## Velog / blog short intro

Linux, 네트워크, AWS 기초를 공부하면서 "명령어를 실행했다"에서 끝나지 않고 결과를 기록으로 남기는 작은 CLI 도구를 만들고 있습니다. 이름은 Ops Study Maintainer Kit입니다. `grep`/`find`, `ping`/`curl`, AWS S3 evidence template처럼 초보자가 자주 헷갈리는 주제를 작은 실습으로 정리합니다. v0.3.0에서는 `ops-study-kit new-evidence-pack` 기능을 추가해서 목표, 환경, 명령어, 관찰 결과, 검증, 배운 점, 포트폴리오 요약을 Markdown으로 만들 수 있게 했습니다. `ops-study-kit list-resources`로 현재 제공되는 docs, labs, templates도 확인할 수 있습니다. 아직 초기 프로젝트지만 Linux, 네트워크, AWS 기초를 공부하는 사람이 실습을 설명 가능한 기록으로 남기는 데 초점을 두고 있습니다. 피드백을 받고 싶습니다: https://github.com/william21c1210-ops/ops-study-maintainer-kit

## GitHub Discussions / community short intro

Ops Study Maintainer Kit은 초보 DevOps/SRE 학습자가 Linux, 네트워크, AWS 실습을 reviewable Markdown evidence로 남기도록 돕는 작은 CLI 도구입니다. 현재 `grep`/`find`, `ping`/`curl`, AWS S3 evidence template, 한국어 quickstart, 일본어 report template을 포함하고 있습니다. v0.3.0에서는 `new-evidence-pack` 명령을 추가해 목표, 환경, 명령어, 관찰 결과, 검증, 헷갈린 점, 배운 점, 포트폴리오 요약을 한 파일로 생성할 수 있게 했습니다. 민감한 cloud 정보는 남기지 않는 방향으로 문서화 예시를 구성했습니다. 초보자 관점에서 더 필요한 실습이나 문서가 있다면 피드백을 받고 싶습니다: https://github.com/william21c1210-ops/ops-study-maintainer-kit
