# Hi, I'm building beginner-friendly DevOps/SRE learning tools.

I'm currently learning Linux, networking, cloud, and DevOps/SRE fundamentals.

My current OSS project:

- [Ops Study Maintainer Kit](https://github.com/william21c1210-ops/ops-study-maintainer-kit)

It helps beginner learners turn command-line practice into reproducible technical evidence through small Linux, networking, and cloud labs.

Current focus:

- Linux and networking practice labs
- AWS S3 beginner evidence templates
- Korean/Japanese beginner-friendly documentation
- CLI helpers for study reports and evidence packs

I'm especially interested in practical documentation that helps learners explain what they ran, what they observed, and what they learned.
