# Contributing

Thank you for considering a contribution.

## Good first contributions

- Fix unclear documentation
- Add a small Linux/networking example
- Add a test for CLI behavior
- Improve Korean/Japanese learning templates

## Pull request checklist

- [ ] The change is small and focused
- [ ] README or examples are updated if needed
- [ ] Tests pass with `pytest`
- [ ] The change helps beginner learners build reproducible evidence
